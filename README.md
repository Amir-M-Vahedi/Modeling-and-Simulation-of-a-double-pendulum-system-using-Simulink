# Modeling and Simulation of a double pendulum system using Simulink
## Overview
This project investigates the dynamics of a double pendulum system, a classic example in physics and engineering illustrating chaotic motion under certain conditions. The focus of this study is on deriving the dynamic equations governing the system's motion, finding analytical solutions, and implementing these equations in Simulink for simulation.

## Methodology
1. **Equation Derivation**: Starting with the physical configuration of the double pendulum, the Lagrangian method is used to derive the equations of motion.
2. **Analytical Analysis**: Solve the derived equations analytically to understand the theoretical motion of the pendulum under various initial conditions.
3. **Simulink Simulation**: Implement the equations in Simulink, creating a model that simulates the motion of the double pendulum.
4. **Comparison and Validation**: The analytical and Simulink-generated numerical results are compared to validate the model's accuracy.
